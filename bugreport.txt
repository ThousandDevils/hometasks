Summary - Короткое описание проблемы, явно указывающее на причину и тип бага
Project - Название тестируемого проекта
Component - Название части или функции тестируемого продукта
Version - версия продукта
Description - шаги, окружение, полученный и ожидаемый результат
Severity - Критичность
Priority - Приоритет
Status - Статус бага
Аttachment - прикрепленный файл
important - автор бага, на кого назначен
